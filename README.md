# ObserVacation-John Bryce Project  #3
SPA built with ReactJS and NodeJS.

## Features:

* MVC Architecture
* Modular structure, arranged by feature

## Backend:
* Node.js + Express
* MySQL DB
* JWT for Auth

## Frontend:
* React.js
* Redux + Thunk
* Material Design
